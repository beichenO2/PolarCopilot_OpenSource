# PC-OS Principles

1. Hub Web is the control plane for human answers.
2. No SubAgent in Solo Web mode.
3. Secrets stay out of prompts (use placeholders).
4. Small, focused diffs.
5. Use **pc-os-*** skill names only — do not mix in skills from other monorepos that use a different `pc-*` prefix.
